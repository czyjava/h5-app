# HomeAI APK 功能清单采集

## 产物

- `outputs/homeai功能清单-2026-05-26.xlsx`：后续逐项 check 使用的功能清单。
- `api/mitm-summary.jsonl`：脱敏后的接口形状记录，只保留 Host、Path、字段名和状态码。
- `outputs/render-*.png`：Excel 各 Sheet 顶部区域的渲染验收图。

## 截图清理规则

- `apk/` 原始模拟器截图和 XML 只作为采集过程中的临时证据。
- 每次导出 Excel、脱敏接口日志和必要渲染验收图后，默认清理原始截图/XML，节省磁盘并避免账号画面、二维码等内容误入远端。
- 如需保留某次原始截图证据，需要在任务开始或结束前明确说明。

## 覆盖范围

- 首页、设计、发现、我的、会员、邀请、问卷、设置等主链路。
- 真实支付、真实生成、问卷提交、反馈提交、退出登录、资料保存/注销、系统权限授权和第三方分享未执行，已统一放入 Excel 的 `待复核动作` Sheet。
